# Icon Assets

Place your app icons here:

- `icon.png` - 1024x1024 PNG (Linux)
- `icon.ico` - Windows icon
- `icon.icns` - macOS icon
- `tray/icon-normal.png` - System tray icon (normal state)
- `tray/icon-alert.png` - System tray icon (alert state)
- `tray/icon-working.png` - System tray icon (working state)

For now, you can use placeholder icons. Replace these with actual SysCat branding later.
